Análise do erro:

Percebe-se que o método RK-4 é bastante superior em relação aos demais, enquanto que o método de Euler é o pior. Os outros 3 métodos são
bem parecidos entre si. Nota-se também que para n razoavelmente grande, o único método que destoa (negativamente) é o de Euler.

Também podemos perceber que analisando apenas o erro no final (o "último" ponto), o método de Heun apresenta certa vantagem sobre os outros
e passa a ser o segundo melhor, perdendo apenas para RK4.

Considerando o erro como a integral do módulo da diferença das funções, voltamos a conclusão feita inicialmente: RK4 o melhor; Euler o pior
e os outros 3 "na média".
